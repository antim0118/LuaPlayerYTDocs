## Обзор
LPYT (LuaPlayerYT, LuaPlayer By YuliaTeam) — это 2D движок и интерпретатор Lua-скриптов для PlayStation Portable
(PSP), разработанный для создания игр и приложений на PSP с использованием языка программирования Lua.

## Основные характеристики
<DotList>
  <>Движок ориентирован на 2D графику и поддерживает работу с файлами напрямую на PSP.</>
  <>Позволяет создавать и запускать игры и приложения, написанные на языке Lua.</>
  <>Обеспечивает поддержку видео, позволяя запускать видеофайлы в играх или приложениях.</>
</DotList>


## Назначение и использование
<DotList>
  <>Главная цель LPYT — упростить разработку игр на PSP без необходимости глубоко знать низкоуровневое программирование.</>
  <>Позволяет создавать 2D игры, визуальные новеллы и мультимедийные проекты, используя простой и понятный скриптовый язык Lua.</>
  <>Подходит для хобби-разработчиков и инди-команд, работающих с ограниченными ресурсами PSP.</>
  <>Может быть использован вместе с другими библиотеками и фреймворками, расширяя возможности разработки.</>
</DotList>
LPYT представляет собой мощный и удобный инструмент для создания 2D игр и приложений на PSP с помощью языка Lua,
сочетая легкость разработки с широкими функциональными возможностями на устаревшей, но популярной платформе.

## Великая история LPYT
Полная история LPYT (LuaPlayerYT) начинается с появления оригинального Lua Player, разработанного для
PlayStation Portable (PSP) как интерпретатор языка Lua, предназначенный для создания игр и приложений на этой
платформе. Lua Player позволял запускать 2D игры и мультимедийные приложения, используя легковесный скриптовый
язык Lua.

Со временем, из-за ограничений первоначального Lua Player и необходимости улучшить функциональность и поддержку
современных возможностей, был создан проект LuaPlayerYT (LPYT). Он стал расширенной и улучшенной версией Lua
Player, включающей:
<DotList>
  <>Поддержку видео воспроизведения через собственный плеер на PSP.</>
  <>Улучшенную обработку графики и звука.</>
  <>Усиленную стабильность и расширенные возможности для запуска сложных 2D проектов.</>
  <>Переписанную и оптимизированную архитектуру для более эффективной работы на PSP.</>
</DotList>

LPYT активно поддерживается и развивается командой Yulia Team, которые адаптируют его для портирования
визуальных новелл и других интерактивных приложений с PC на PSP. Примером такого проекта является порт
визуальной новеллы "Бесконечное Лето" (Everlasting Summer), где с помощью LPYT реализована переадаптация
контента с RenPy на Lua, что позволило играть в новеллу на PSP.

LPYT продолжает поддерживаться и обновляться с 2022 года, в основном в рамках сообщества хобби-разработчиков
и инди-команд, заинтересованных в разработке игр для PSP с использованием Lua.

## Основные возможности и API LPYT
<DotList>
  <><b>Интерпретатор Lua</b>: LPYT запускает Lua-скрипты, позволяя управлять логикой игры, взаимодействием с пользователем и обработкой событий.</>
  <><b>Графика 2D</b>: API позволяет рисовать спрайты, текст и простые фигуры на экране.</>
  <><b>Воспроизведение аудио и видео</b>: Поддержка воспроизведения аудио (WAV, MP3, AT3) и видео (PMP).</>
  <><b>Обработка ввода</b>: Поддержка управления с кнопок PSP, включая аналоговый стик и клавиши.</>
  <><b>Управление ресурсами</b>: API предоставляет функции для загрузки и управления графическими и аудиофайлами из памяти PSP.</>
  <><b>Оптимизация использования памяти</b>: Инструменты и функции для эффективного использования ограниченной памяти PSP без утечек.</>
</DotList>

Эти возможности дают разработчикам простой и мощный инструмент для реализации 2D игр, визуальных новелл и
мультимедийных приложений на этой платформе.
